# template-koa2

> SAO template for koa2

## Usage

Install [SAO](https://github.com/egoist/sao) first.

### From git

```bash
$ sao zcong1993/template-koa2 [your project name] --install
```

### From npm

```bash
$ npm install -g template-koa2
$ sao koa2
```
